**ArkadiaQuest** is a powerful and intuitive tool that allows anyone to create Arkadia quests with ease.

Whether you're designing simple storylines or complex quest in-game, ArkadiaQuest streamlines the entire creation process.

---

## ⭐ Key Features

ArkadiaQuest is structured around two main modules:
| Module | Description |
| ----------- | ---------- |
| QuestEditor | Create, edit, and organize quests efficiently. |
| NpcEditor | Manage all NPCs and their related data in one centralized place. | 

## 📜 Quest Editor

This section gathers all the current quests, organized into groups.

You can enter the edit mode of a quest by clicking on it.

<!-- Screenshot here -->

## 🧙 NpcEditor

Here you will find the list of all NPCs available on Arkadia. You can add various pieces of information such as `skin texture`, `name`, `coordinates`, `custom lore` and much more. 

<!-- Screenshot here -->

This tool helps our scribes keep all NPCs stored in one centralized place. Additionally, when creating a new quest, users can easily select existing characters instead of spending time searching through the lore.

> [!WARNING]
> This app is **only** compatible with [BeautyQuest](https://modrinth.com/plugin/beautyquests) using [LuxDialogue](https://builtbybit.com/resources/luxdialogues-interactive-dialogues.60954/) plugins.

## 🛠️ Usage

### Installation

Install and launch dev environment using :
```bash
git clone https://github.com/Game-Of-Arkadia/App-ArkadiaQues
cd ArkadiaQuest
npm install
npm run dev
```
*Access the app at `http://localhost:8080`.*

To be able to send requests for the NPCs skin, launch the backend by running :
```
npm run server
```
